Data Analytics Project

This project demonstrates a complete Data Analytics workflow using Python, MySQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL analysis, and interactive visualizations.

The project follows an end-to-end analytics pipeline that is commonly used in real-world business environments.

Data Loading
Imported the dataset using Python.
Explored the dataset structure and data types.
Exploratory Data Analysis (EDA)
Analyzed data distributions and relationships.
Identified trends, patterns, and potential issues.
Generated summary statistics and visualizations.
Data Cleaning
Handled missing values.
Removed duplicate records.
Corrected data types.
Treated inconsistent or invalid values.
Prepared the dataset for analysis.
SQL Analysis (MySQL)
Imported the cleaned dataset into MySQL.
Wrote SQL queries to answer business questions.
Power BI Dashboard
Connected Power BI to the processed dataset.
Built an interactive dashboard.
